## Provider Utilization & DRG Financial Impact Dashboard

### Overview
This Power BI dashboard analyzes Medicare inpatient utilization and DRG‑level financial impact, with a focus on identifying the highest‑cost DRGs and understanding provider performance. The report is designed for executive‑level consumption, using clean visuals, modern formatting, and healthcare‑specific domain logic.

The dashboard highlights Medicare spend in the **billions**, providing a realistic view of inpatient financial scale and cost drivers.

---

## Key Features
- **Top DRGs by Medicare Spend (Billions)**  
  Clean column visual showing the highest‑cost DRGs with executive‑grade formatting.
- **Provider Utilization Metrics**  
  Summary visuals showing counts, totals, and utilization patterns.
- **Interactive Slicer Panel**  
  Filter by DRG, MDC, provider, and other attributes.
- **Modern, Minimalist Layout**  
  Axis‑free visuals, clean typography, and consistent color palette.
- **Healthcare Domain Alignment**  
  Uses DRG logic, Medicare payment fields, and inpatient financial concepts.
